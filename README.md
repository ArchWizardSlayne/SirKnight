# SirKnight

A 2D platformer game built with Godot 4.5.

## Features

- Player movement with run, jump, and idle animations
- Collectible coins with score tracking
- Enemy AI (slimes, flying demons)
- Multiple level design with hazards and goal objectives
- Pixel art aesthetic

## Controls

- **A / D** - Move left/right
- **Space** - Jump

## Technical Skills Demonstrated

- Godot 4.5 game engine
- GDScript programming
- Character physics and collision detection
- State-based animation systems
- Level design and scene management

## Running the Project

1. Install [Godot 4.5](https://godotengine.org/)
2. Open the project in Godot
3. Press F5 to run
